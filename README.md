# udacity-embedded-3.11-interrupt

## Exercise Overview
This exercise will put multiple things from this lesson together using a Wokwi simulation. In this exercise, you will write the code for the main program to toggle an LED every 1 second (1000ms). You will also write the code for an external interrupt to toggle the state of the buzzer when the connected button is pushed 3 times.

Note : Tiny85 ( AVR ATMega8 ) will again be used for this exercise

The exercise simulation can be accessed from this linkhere(opens in a new tab).

In the connected circuit,

LED is connected to PB1
Buzzer is connected to PB0
Button is connected to PB2
Instructions :
Write the code for main to toggle LED every 1000ms .
Configure external interrupt 0 as follows:
Enable Global interrupt.
Configure External interrupt 0 as Rising Edge.
Enable External interrupt 0.
Write a code for External interrupt 0 to toggle Buzzer state when External interrupt 0 is triggered for 3 times.
Test your code.
Note : When the buzzer state is changed, you will see this sign ♫.
